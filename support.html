Translation & language learning for macOS

Glotty (歌蒂) lives in your menu bar. Select text in any app, hold the Fn key, and press one key to act on it.

Getting started

  1. On first launch, grant the two permissions Glotty asks for in System Settings → Privacy & Security: Accessibility (to read your selected text) and Input Monitoring (for the keyboard shortcut). Glotty's in-app Permissions screen guides you through this.
  2. Select any text, then hold Fn and press:
    • T — Translate (works with no setup)
    • E — Explain the meaning and grammar
    • P — Polish your writing
    • C — Chat with a language tutor

Setting up AI features (Explain / Polish / Chat)

These use an AI provider that you connect with your own API key. Open Settings → Language Model, choose a provider (such as OpenAI, DeepSeek, or Grok), and paste your key. Translate does not need this — it works out of the box.

Troubleshooting

  • Pressing the shortcut does nothing — check that Accessibility and Input Monitoring are enabled for Glotty in System Settings → Privacy & Security, then try again.
  • Explain / Polish / Chat show no result — make sure you've added an AI provider and API key in Settings → Language Model.
  • Nothing happens after selecting text — make sure the text is actually selected (highlighted) before holding Fn.
